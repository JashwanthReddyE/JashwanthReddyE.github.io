AISLakehouse

      
aisstream.io ─▶ event hubs ─▶ spark streaming ─▶ delta bronze→gold ─▶ dashboard

      
A round-the-clock streaming lakehouse tracking global ship traffic — and flagging vessels that go dark.

      

        Live ship positions stream over a persistent WebSocket into Azure Event Hubs, then through Spark Structured Streaming
        into a bronze → silver → gold Delta lakehouse on Databricks + ADLS Gen2. The gold layer does the interesting part:
        dark-vessel detection, tanker flow through the Strait of Hormuz, and floating-storage signals —
        behavior analytics, not dots on a map.
      

      
    
        
  • Exactly-once into bronze via Spark checkpointing + Delta transactional writes over an at-least-once producer
    • 
        
  • 24/7 containerized ingestion on Azure Container Apps, provisioned with Bicep IaC — budget alert created before any billable resource
    • 
        
  • Hourly GitHub Actions refresh re-runs bronze → gold and redeploys the live dashboard automatically
    • 
        
  • WebSocket lifecycle done right: exponential backoff + jitter, idle heartbeat, graceful shutdown — all unit-tested
    •

Repo Raconteur

      
github tarball ─▶ ingest + filter ─▶ foundry iq index ─▶ grounded narration ─▶ cited story

      
Turns any GitHub repository into grounded interactive fiction — every plot beat cited back to real source code.

      

        Point it at a codebase. It indexes the files into a Foundry IQ knowledge base on Azure AI Search,
        casts the most central modules as characters, and Azure OpenAI narrates a heist, noir, or fantasy story.
        The narrator only ever sees retrieved snippets — so the story literally cannot reference code that isn't in the index.
        Built for Microsoft's AI Skills Fest (Creative Apps with GitHub Copilot).
      

      
    
        
  • Citations correct by construction — the retrieval that grounds each beat IS the citation; click any one to see the exact source
    • 
        
  • Three genre-reactive UIs: palette, typography, and ambient canvas animation all shift with the story's mood
    • 
        
  • Graceful degradation — with a blank .env it runs fully local (in-memory index + built-in narrator); never hard-fails
    • 
        
  • Dependency-graph analysis picks the protagonist and supporting cast from real imports
